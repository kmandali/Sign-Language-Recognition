# Real-Time Sign Language Recognition Django App

A real-time Sign Language Recognition web application built using **Django**, **TensorFlow**, and **Convolutional Neural Networks (CNNs)**. The system is capable of recognizing and classifying up to **10 different sign language gestures** through a user-friendly web interface.

The application integrates:
- **Frontend:** HTML/CSS for an interactive and responsive user interface
- **Backend:** Django framework for server-side processing
- **Machine Learning:** CNN-based deep learning model for gesture recognition

---

## Features

### Real-Time Gesture Recognition
Uses a trained CNN model to detect and classify sign language gestures in real time.

### Multi-Class Classification
Supports recognition of up to **10 different sign language labels**.

### User-Friendly Interface
Designed with HTML and CSS to provide a simple, clean, and intuitive user experience.

### Full-Stack Implementation
Includes:
- Frontend Development
- Backend Integration
- Machine Learning Model Deployment

## Installation & Setup

### 1. Clone the Repository

Clone this repository to your local machine using:

```bash
git clone https://github.com/your-repo/sign-language-recognition.git
```

---

### 2. Install Required Dependencies

Navigate to the project directory and install the required Python packages:

```bash
pip install -r requirements.txt
```

---

### 3. Run the Django Development Server

Start the Django server using:

```bash
python manage.py runserver
```

---

### 4. Open the Application

After the server starts, open your browser and visit:

```bash
http://127.0.0.1:8000/
```
